# 🚀 Namma-Skill

**Namma-Skill** is an Android application designed to connect youth in small towns with nearby skill development training programs. It helps users discover short-term courses and never miss batch enrollments.

---

## 📌 Problem Statement

Many government and private skill development centers offer valuable short-term courses (like welding, tailoring, mobile repair, etc.), but:

- Students are unaware of upcoming batches
- There is no centralized platform for updates
- Opportunities are missed due to lack of information

---

## 💡 Solution

Namma-Skill solves this problem by providing:

- 📚 A list of available skill courses
- 🔍 Search functionality to find courses بسهولة
- 🎯 Filters to refine results (category, duration, etc.)
- 📅 Timely updates on new batches
- 📍 Location-based discovery (future scope)

---

## 📱 Features

- 🔎 Search courses by name
- 🗂️ Filter courses by category
- 📄 View course details
- 📢 Updates on new training batches
- 📱 Clean and user-friendly UI
